## 一、项目简介
本项目为计算机图形学课程实验作业，基于 Python 实现轻量化实时粒子系统。模拟大量粒子的生成、运动、受力、消散与重生过程，复现自然界烟雾、流光、散点飞舞等动态视觉效果。
项目代码高度精简、无冗余封装，完整保留粒子系统核心数学逻辑，适合用于理解图形学动态模拟、随机生成、物理运动模拟基础原理。

## 五、核心算法原理（重点）
本项目粒子系统遵循计算机图形学动态粒子模拟标准流程：粒子生成 → 状态更新（受力/位移） → 边界检测 → 生命周期消亡/重生 → 逐帧渲染。
1. 粒子基本属性定义
每一个粒子为独立对象，包含以下状态参数：
- 坐标位置：(x, y)
- 运动速度：(vx, vy)
- 加速度/扰动：模拟风力、重力随机偏移
- 生命周期：存活帧数，超时自动销毁
- 外观属性：大小、透明度、颜色渐变
2. 粒子生成策略
程序初始化与运行过程中持续批量生成新粒子：
- 在屏幕指定中心区域随机生成初始位置；
- 速度采用随机向量，保证粒子发散飞舞效果；
- 初始生命周期随机，避免所有粒子同步消亡，视觉更自然。
3. 粒子运动物理迭代（核心公式）
每一帧对所有粒子进行运动状态更新，基于匀速运动+随机扰动模型：
$$x_{new} = x_{old} + v_x$$
$$y_{new} = y_{old} + v_y$$
同时加入微小随机扰动，模拟自然飘动效果，避免机械匀速运动的僵硬感。
速度随生命周期缓慢衰减，实现粒子后期缓缓消散的视觉效果。
4. 边界碰撞重置机制
当粒子运动超出画布宽高边界时，不做反弹处理，直接重置位置与速度，实现粒子循环常驻、持续动态效果，保证画面不会出现粒子大面积消失、空洞黑屏问题。
5. 生命周期与消亡机制
- 每一帧所有粒子生命周期计数 -1；
- 生命周期趋近于 0 时，降低粒子透明度与大小，实现渐变消失；
- 生命周期归零后销毁该粒子，并生成新粒子替补，维持画面粒子总数稳定。
6. 逐帧渲染逻辑
采用帧动画循环机制：清空画布 → 更新所有粒子物理状态 → 批量绘制所有粒子 → 刷新窗口。保证动态画面流畅、连贯。
## 六、效果特点
- 随机性强：位置、速度、生命周期随机，动态效果自然不重复；
- 物理模拟真实：速度衰减、随机扰动、渐变消散贴合真实粒子运动；
- 性能轻量化：纯 Python 实现，无重型引擎依赖，运行流畅；
- 可扩展性高：可快速修改为星空、烟雾、萤火、雪花等特效。
## 七、自定义修改说明
可直接在源码中修改以下参数，自定义特效：
- 粒子最大数量：控制画面疏密程度
- 粒子运动速度：调整飞舞快慢
- 生命周期：调整粒子留存时长
- 粒子大小、颜色、透明度：更改视觉风格
- 随机扰动强度：控制飘动剧烈程度
